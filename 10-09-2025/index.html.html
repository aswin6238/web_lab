What is the Web?

The Web is a collection of documents and applications you open in a browser. it is build using three main technologies: HTML, CSS, and Javascript.

Core Technologies

  2. Css - Design

  3. Javascript - Interaction

Why They Matter

Together, these three form the backbone of every modern website:

  • Html: provides structure (headings, paragraph, images)
  • Css: Defines Style (colors, layout, fonts)
  • Javascript: Add Interactivity and dynamic behavior
